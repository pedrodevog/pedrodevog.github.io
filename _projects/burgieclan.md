## Overview

Burgieclan is a student platform built to make sharing study materials easier and to support collaborative exam preparation.

## Repository

The source code is available on [GitHub](https://github.com/VTKLeuven/burgieclan).

## Focus Areas

- Student collaboration and study material sharing
- Modern web application development
- Production-oriented deployment with Docker and GitHub Actions

## Notes

The project combines a frontend (NextJS) and backend (PHP Symfony) stack and is designed for a real user community.
